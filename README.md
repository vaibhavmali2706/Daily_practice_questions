# Daily Practice Questions

A collection of Data Structures and Algorithms (DSA) solutions solved through regular practice.

This repository serves as a personal reference for algorithmic patterns, problem-solving techniques, and clean Python implementations. It is maintained as part of continuous learning and is updated regularly as new problems are solved.

---

## Objectives

- Maintain consistency in DSA practice.
- Strengthen problem-solving skills.
- Build familiarity with common algorithmic patterns.
- Document solutions for future reference.
- Improve code quality and analytical thinking.

## Solution Format

Whenever applicable, each solution aims to include:

- Problem name
- Python implementation
- Time Complexity
- Space Complexity
- Brief explanation or approach

Example:

```
Problem:
LeetCode 217 - Contains Duplicate

Approach:
HashMap / HashSet

Time Complexity:
O(n)

Space Complexity:
O(n)
```

---

## Coding Principles

The solutions in this repository generally follow these principles:

- Prioritize correctness before optimization.
- Improve time and space complexity whenever possible.
- Write clean and readable code.
- Use appropriate data structures based on the problem.
- Learn multiple approaches (Brute Force → Better → Optimal) where applicable.
